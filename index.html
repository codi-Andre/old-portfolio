Sobre

Comecei a estudar programação com JavaScript, HTML e CSS, após ter construído uma base nestas linguagens 'core', passei para ferramentas mais complexas.

Pude contar com a orientação e críticas de um desenvolvedor sênior, que ensionou-me conceitos mais complexos como, versionamento de código com git e github e noções básicas de UI/UX.

Atualmente, pratico a construção de aplicações web utilizando ReactJS, Typescript e NextJS e também, estudo algoritmos e estrutura de dados utilizando a plataforma 'leetcode'.

Portfolio

Projeto em destaque

Este portfolio foi desenvolvido com a metodologia 'mobile first' em mente, assim como semântica e acessibilidade, sendo responsivo em diferentes dispositivos. Também fazendo o uso da convenção de nomes 'BEM' para classes, priorizando por um código limpo e de fácil manutenibilidade.

Por ser um projeto pequeno, visando sua longevidade e reduzir a dependência de fontes externas, não foi utilizado bibliotecas de terceiros, apenas a ferramenta Vite para gerar uma versão de produção reduzindo o tamanho final do código.
